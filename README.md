## Немного про работу этой утилиты
Данный модуль является оболочкой над всей экосистемой plg (python love gmod). Он (этот модуль) занимается инкрементальной компиляцией и в принципе менеджерит все настройки и сборки модулей входящих в себя

## Модули sdk
Текущие используемые модули для данного sdk:
- [py2glua](https://github.com/py-love-gmod/py2glua) - компилятор питоновского кода в глуа код.
